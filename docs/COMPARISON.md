### Quick Configuration

Add to your `.github/workflows/smartbrain.yml`:

```yaml
name: SmartBrain Automation
on: [push, pull_request]

jobs:
  smartbrain:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Run @SmartBrain
        uses: smartbrain/action@v1
        with:
          audit: true
          gas-optimization: true
          auto-deploy: testnet
```
